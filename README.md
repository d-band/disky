Disky
=====

> Make NTFS writable on macOS

![Disky](images/image.png)

## Development

```bash
npm i dool -g
npm i

npm run dev
npm run start

# Electron Mirror of China
ELECTRON_MIRROR="https://npm.taobao.org/mirrors/electron/"

npm run pack
npm run disk
```
